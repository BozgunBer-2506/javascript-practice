# JavaScript Practice - Arrays, Objects & ES6 Modules

## Description

This project covers fundamental JavaScript concepts including:
- Arrays and array methods (map, filter, find)
- Objects and destructuring
- Spread operator
- ES6 modules (import/export)
- Immutability principles

## Parts
1. Arrays Basics
2. Array Methods
3. Spread Operator
4. Destructuring
5. map()
6. filter()
7. find()
8. Chaining
9. Objects
10. Spread Objects
11. Object Destructuring
12. Immutability
13. ES6 Modules
14. Final Project

## How to Run

```bash
node 1_arrays_basics.js
node 2_array_methods.js
node 3_spread_operator.js
node 4_destructuring.js
node 5_map.js
node 6_filter.js
node 7_find.js
node 8_chaining.js
node 9_objects.js
node 10_spread_objects.js
node 11_object_destructuring.js
node 12_immutability.js

# For ES6 Modules
cd 13_modules
node app.js

# Final Project
cd ..
node 14_final_project.js
```
